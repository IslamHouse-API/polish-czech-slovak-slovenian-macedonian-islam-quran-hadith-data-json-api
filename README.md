## Repository overview

This repository groups five Slavic language channels into one structured access layer:

- Polish
- Czech
- Slovak
- Slovenian
- Macedonian

Its purpose is clear: give this linguistically coherent cluster a focused and practical place for discovery, integration, and multilingual Islamic content workflows.

---

## Why this repository exists

This cluster is built on real linguistic proximity.

Polish, Czech, and Slovak form a closely related West Slavic core, while Slovenian and Macedonian extend the repository into South Slavic environments — all within the same broader Slavic language family.

This repository creates a cleaner layer for:

- regional discoverability
- structured Islamic content access
- multilingual publishing workflows
- localization and deployment
- educational product building
- cleaner repository segmentation
